# سیستم مدیریت نمرات دانش‌آموزان

یک برنامه پایتونی برای مدیریت نمرات دانش‌آموزان در سه درس (ریاضی، علوم، انگلیسی).

## قابلیت‌ها
- نمایش همه دانش‌آموزان و نمرات
- اضافه کردن دانش‌آموز جدید
- محاسبه معدل هر دانش‌آموز
- پیدا کردن بهترین دانش‌آموز
- تغییر نمره یک درس خاص

## نحوه اجرا
```bash
python contacts_manager.py
