## 🎮 ۱. بازی حدس عدد (Guess the Number)

یک بازی سرگرم‌کننده با سطوح مختلف!

**قابلیت‌ها:**
- ۳ سطح: آسان (۱-۵۰)، متوسط (۱-۱۰۰)، سخت (۱-۲۰۰)
- راهنمای دما 🔥 (خیلی گرم، گرم، سرد، یخ)
- حداکثر ۱۰ بار شانس
- نشون دادن مسیر حدس‌ها

**اجرا:**
python guess_number.py

## 📊 ۲. سیستم مدیریت نمرات

مدیریت نمرات دانش‌آموزان در سه درس.

**قابلیت‌ها:**
- نمایش همه نمرات
- اضافه کردن دانش‌آموز جدید
- محاسبه معدل
- پیدا کردن بهترین دانش‌آموز
- تغییر نمره

**اجرا:**
python contacts_manager.py
